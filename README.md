# (주)새론 재고관리 시스템

## 📦 프로젝트 소개

(주)새론을 위한 종합 재고관리 시스템입니다. 실시간 재고 현황 조회, 입출고 관리, 통계 분석 등 전문적인 재고 관리 기능을 제공합니다.

## ✨ 주요 기능

### 📊 대시보드
- 실시간 재고 현황 요약
- 카테고리별 재고 분석
- 부족 재고 알림
- 입출고 통계
- 재고 가치 계산

### 📦 재고 관리
- 제품 등록/수정/삭제
- 실시간 입고 처리
- 실시간 출고 처리
- 제품 검색 및 필터링
- 재고 상태 모니터링
- 카테고리별 분류

### 📝 입출고 내역
- 전체 거래 내역 조회
- 입고/출고 필터링
- CSV 내보내기
- 거래 통계

### 📈 리포트
- 카테고리별 재고 분석
- 입출고 추이 그래프 (30일)
- 상위 제품 분석
- 재고 가치 분석

### ⚙️ 설정
- 데이터 백업 (JSON)
- 데이터 복원
- 데이터 초기화
- 시스템 정보

## 🚀 시작하기

### 필수 요구사항
- Node.js 18.x 이상
- npm 또는 yarn

### 설치 방법

1. 의존성 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm run dev
```

3. 브라우저에서 접속
```
http://localhost:3000
```

### 빌드

프로덕션 빌드:
```bash
npm run build
```

빌드 미리보기:
```bash
npm run preview
```

## 🛠️ 기술 스택

- **Frontend**: React 18
- **Routing**: React Router v6
- **State Management**: Zustand (with persist middleware)
- **Charts**: Recharts
- **Icons**: React Icons
- **Date Handling**: date-fns
- **Build Tool**: Vite
- **Styling**: Inline CSS (Component-based)

## 📱 주요 화면

### 대시보드
- 총 제품 수, 재고 수량, 부족 재고, 총 재고 가치 카드
- 카테고리별 재고 수량/가치 차트
- 최근 7일 입출고 현황
- 부족 재고 알림 리스트

### 재고 관리
- 제품 목록 테이블
- 실시간 검색 및 카테고리 필터
- 입고/출고 버튼으로 빠른 처리
- 제품 상태 표시 (정상/부족/품절)

### 입출고 내역
- 전체 거래 내역 타임라인
- 입고/출고 구분 필터
- CSV 다운로드 기능

### 리포트
- 카테고리별 재고 분석 (막대 그래프)
- 재고 가치 분포 (파이 차트)
- 30일 입출고 추이 (라인 차트)
- 상위 10개 제품 분석

## 💾 데이터 저장

- **로컬 스토리지**: 브라우저 로컬 스토리지를 사용하여 데이터 영구 저장
- **자동 저장**: 모든 변경사항 자동 저장
- **백업/복원**: JSON 파일로 백업 및 복원 가능

## 📋 샘플 데이터

시스템에는 다음과 같은 샘플 데이터가 포함되어 있습니다:
- 5개의 샘플 제품 (노트북, 마우스, USB, 모니터, 키보드)
- 3개의 샘플 거래 내역
- 4개의 카테고리 (전자제품, 주변기기, 저장장치, 기타)

## 🔒 데이터 보안

- 클라이언트 사이드 저장소 사용
- 민감한 정보는 저장하지 않음
- 정기적인 백업 권장

## 📝 사용 가이드

### 제품 등록
1. 재고 관리 페이지 접속
2. "제품 추가" 버튼 클릭
3. 제품 정보 입력
4. "추가" 버튼으로 저장

### 입고 처리
1. 재고 관리 페이지에서 제품 찾기
2. 입고 버튼 (↑) 클릭
3. 수량, 담당자, 비고 입력
4. "입고" 버튼으로 처리

### 출고 처리
1. 재고 관리 페이지에서 제품 찾기
2. 출고 버튼 (↓) 클릭
3. 수량, 담당자, 비고 입력
4. "출고" 버튼으로 처리

### 데이터 백업
1. 설정 페이지 접속
2. "데이터 내보내기" 클릭
3. JSON 파일 다운로드
